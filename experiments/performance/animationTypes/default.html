Animation Types

There are two main categories for DOM animations: dependent and independent.

Dependent animations use the CPU but provide a great deal of flexibility and variability. They also can use less memory than independent animations.

Independent animations use the GPU to animate the item. The layer is pushed to the GPU along with the animation and the GPU performs the work for the animation.Independent animations can take more memory because they get their own layer/surface. However, the have the benefit of freeing up the CPU to do other important work.
