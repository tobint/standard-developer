CSS Gradients: Abused

CSS Gradients can do some powerful things. Some people dont't realize that you can apply multiple gradients to a single element. This results in a surface with the combination of all gradients.

Select a photo using the file picker (it doesn't upload anywhere). When the fans on your graphics card cool down, you'll get a CSS-rendered photo using thousands of radial gradients. This works by drawing the photo you select to a canvas in the browser, and then reading each pixel in the canvas to determine what parameters to add for the radial gradient.
