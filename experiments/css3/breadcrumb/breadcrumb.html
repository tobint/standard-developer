CSS Breadcrumb

CSS 2D transforms + gradients + overflow = CSS breadcrumb

CSS styled breadcrumbs have been around for a while. However, getting a non-standard separator can be very tricky, particularly if you want the void of a separator to change color along with the highlighted stage in the breadcrumb.

This is one of a few ways to pull this off without using images. It leaves you open to completely modify the color scheme without worry about updating graphics.
